\doxysubsection{Detailed Description}
Command-\/line interface for testing multiple knapsack algorithms. 

Allows users to choose between brute-\/force, dynamic programming, greedy approximation, and ILP-\/style approaches to solve instances of the 0/1 knapsack problem. Reads dataset files and reports results including selected pallets, profit, and execution time.
